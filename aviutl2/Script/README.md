# Aviutl2用スクリプト

Created by 錠罪

# 導入方法
起動しているAviutl2の**再生プレビュー上に** .anm2ファイルをドラッグアンドドロップまたは、\
`C:\ProgramData\aviutl2\Script\` に .anm2ファイルを移動してください。

# 使い方

## おっおースクリプト Ver.2.0
### テンポ
跳ねるアニメーションの速さをテンポ(BPM)で指定します。\
スクロールの速さにも影響します。
### ハネ量
跳ねるアニメーションの最大変化量を指定します。\
マイナスで上方向, プラスで下方向
### 大きさ加算
跳ねるアニメーション中の拡大率を**パーセントで**指定します。\
10に指定→最大で元の大きさの1.1倍に変化する
### スクロール拍数
跳ねるアニメーション何回分で1スクロールするかを決めます
### スクロール方向
スクロールする方向を指定します。\
`なし`：スクロールしない\
`通常`：左に向かってスクロール\
`逆方向`：右に向かってスクロール

## かんたん複窓動画
### 一辺の数
縦横に何個の動画が連ねるかを指定します。\
100窓動画の場合→10\
プログラムで現れすなら `math.ceil(math.sqrt(DougaNoKazu))`
### 動画No
左上から数えて何番目の動画化を指定します。\
![左から右、上から下の順で番号が変わります。](https://jo-zai.github.io/sort.png)
> [!NOTE]
> 最初は0から始まります。
### 動画Noをobj.indexで置き換える
有効化すると個別オブジェクトの固有値を動画Noとして代入します。\
個別オブジェクトとして複数の動画を読み込めるスクリプトがあれば、この項目で配置を自動化できます。（理論上は）

## 一点配置
### 個数
個別オブジェクトとして何個配置するかを指定します。

## 座標を向く
### X
オブジェクトを向ける座標の横軸上の位置を絶対座標で指定します。
### Y
オブジェクトを向ける座標の縦軸上の位置を絶対座標で指定します。
