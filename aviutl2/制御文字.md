# Aviutl2で使える制御文字一覧
## 色変更
```
<#FFFFFF>白のテキスト<#>
```

## サイズ変更
```
<s60>サイズ60のテキスト<s>
```

## フォント変更
```
<@Comic Sans MS>コミックサンズのテキスト<@>
```

## 表示速度変更
```
<r10>表示速度10で表示されるテキスト<r>
```
> [!NOTE]
> 指定する数値は「表示速度」のパラメータと同じです。

## 表示ウェイト
```
先に表示されるテキスト<w3>3秒待ってから表示されるテキスト
```

## 表示クリア
```
1秒後に消えるテキスト<c1>消えてから表示されるテキスト
```

## 座標指定
```
<p+10,+20>X軸上に10 Y軸上に20ずれるテキスト<p>
```
正負符号なしで揃え位置を基準にした絶対座標で指定
```
テキスト<p0,100>下に表示されるテキスト
```

## 太字
```
<@+B>太字で表示されるテキスト<@-B>
```
**太字はこんな感じです**\
フォントによっては太字に対応していない場合があります。

## 斜体
```
<@+I>斜体（傾いた文字）で表示されるテキスト<@-I>
```
_斜体はこんな感じです_

## 取り消し線
```
<@+S>取り消し線が引かれるテキスト<@-S>
```
~~取り消し線を引くとこんな感じです~~

## プリセット適用
```
<$プリセット名>プリセットが適用されるテキスト<$>
```

## ふりがな
```
</>文章<!>ぶんしょう</>にふりがなをふる
```

## 字間変更
```
<gw10>字間10で表示されるテキスト<gw>
```

## 行間変更

```
<gh10>
行間10で表示される
テキスト
<gh>
```

## 横スケール指定
```
<tw0.5>横幅が0.5倍になるテキスト<tw>
```
## 縦スケール指定
```
<th0.5>縦幅が0.5倍になるテキスト<th>
```

## 角度指定
```
<tr30>30度傾いて表示されるテキスト<tr>
```

## コメント
```
<//この部分は動画に映りません//>
```

## スクリプト
```
<?任意のスクリプト?>
```
例：テキストのフレーム時間を表示する
```
<?mes(obj.frame)?>
```
